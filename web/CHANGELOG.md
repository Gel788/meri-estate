# 📝 Changelog - Meri Movs Web

## [1.0.0] - 2025-12-17

### ✨ Добавлено
- 🎬 Splash screen с анимацией
- 🔑 Анимация ключа и замка
- 📱 Онбординг с 5 слайдами
- 🏠 Каталог недвижимости (8 объектов)
- 🔍 Поиск по названию, адресу, городу
- 🎯 Расширенные фильтры:
  - Тип недвижимости (6 типов)
  - Статус (продажа/аренда)
  - Диапазон цен (слайдеры)
  - Диапазон площади (слайдеры)
  - Количество комнат (0-6)
  - Город
- 📊 Сортировка:
  - По новизне
  - По цене (↑↓)
  - По площади (↑↓)
- ❤️ Избранное с сохранением в localStorage
- 📱 Детальный просмотр объектов
- 🗺️ Интерактивная карта
- 🧮 Калькулятор ипотеки:
  - Расчёт ежемесячного платежа
  - Детализация выплат
  - График распределения
- 👤 Профили агентов
- 📞 Связь с агентами (звонок, email)
- 🎨 Красивые градиентные изображения
- 📱 Адаптивный дизайн
- 💫 Плавные анимации (Framer Motion)
- 🎯 Bottom navigation

### 🎨 Дизайн
- Минималистичный белый фон
- iOS цветовая палитра
- Современные компоненты
- Плавные переходы

### 🚀 Технологии
- React 18
- Vite 5
- Framer Motion 10
